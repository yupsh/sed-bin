# yup-sed

```
NAME:
   sed - stream editor for filtering and transforming text

USAGE:
   sed [OPTIONS] [SCRIPT] [FILE...]

      sed is a stream editor. A stream editor is used to perform basic text
      transformations on an input stream (a file or input from a pipeline).

COMMANDS:
   help, h  Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --expression value, -e value  add the script to the commands to be executed
   --file value, -f value        add the contents of script-file to the commands to be executed
   --in-place, -i                edit files in place (default: false)
   --quiet, -n, --silent         suppress automatic printing of pattern space (default: false)
   --regexp-extended, -r, -E     use extended regular expressions in the script (default: false)
   --help, -h                    show help
```
